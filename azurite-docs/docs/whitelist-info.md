# Whitelist/Intake Proces - Azurite

### Welkom bij het whitelist/intake proces van Azurite! 

Nog nooit op een whitelisted server gespeeld of een intake gedaan? Geen enkel probleem, laat je absoluut niet afschrikken en lees rustig verder. Iedereen is ooit ergens moeten beginnen en misschien is dit voor jou wel het begin!
Onze prioriteit is dat iedereen zich ontzettend thuis voelt bij ons, ervaren of onervaren! Daarom hebben we een kleine samenvatting gemaakt.
In eerste instantie zodat julie precies weten wat er op jullie afkomt, maar nog belangrijker om duidelijk te maken dat een whitelisted server of een intake misschien helemaal niet zo eng hoeft te zijn als jullie misschien denken!
Uiteindelijk zijn wij ook maar gewoon mensen net zoals jij en willen we iedereen een leuke tijd bezorgen terwijl we de roleplaykwaliteit blijven garanderen!

### Wat is een whitelisted server en waarom doen we dit?

Een whitelisted server zorgt ervoor dat je eerst een whitelist oftewel "toestemming" moet krijgen voor dat je de server kan joinen.
Die whitelist oftewel "toestemming" kan je krijgen door een mondelinge intake te doen. 
Tijdens een mondelinge intake zal er onderandere geluisterd worden naar je kennis over de regels. Een intake zal afgenomen worden door Ã©Ã©n of meerdere leden van ons staff/intakers team.
Dit doen we deels voor het roleplayniveau hoog te houden maar zeker ook met de intentie om trollers en hackers uit de server te houden.

### Hou rekening met:
- De duur van een gesprek. We willen natuurlijk dat het bij iedereen zo vlot mogelijk verloopt maar dit is uiteraard niet altijd het geval. Een intakegesprek kan soms namelijk uitlopen door omstandigheden. We verwachten per persoon een intake gesprek van **10 tot 15min**.
- Taalgebruik.
- Aankondiging dat de intakes gesloten zijn. Dit wil zeggen dat er een aankondiging is geplaatst met een **tijdstip** dat de intakes stoppen. Vanaf dit **tijdstip** zal je de **waiting-room** niet meer kunnen joinen en zal je helaas moeten wachten tot een volgend intake moment.
